# LiuTech Device Repair Website

Business website for LiuTech Device Repair services.

## Structure

```
/
├── index.html          # Main website file
├── assets/             # Logo files
│   ├── Logo1.png
│   └── Logo2.png
├── images/             # Service images and photos
│   └── (all image files)
└── .gitignore          # Git ignore file
```

## Deployment

This repository is set up for GitHub Pages. Once pushed to the `main` branch, the site will be available at:
https://liu-tech-ca.github.io

## To Push Changes

1. Make your changes
2. Commit: `git add -A && git commit -m "Your message"`
3. Push: `git push origin main`

## Local Development

Simply open `index.html` in a web browser to view the site locally.
